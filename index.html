A lighter alternative JavaScript View object

JavaScript

When designing a website it is important to reduce HTTP calls as much as possible. One method which helps this is to cache AJAX requests and only fetch them again when the data is changed. If no data has changed, the previously loaded data is on hand to instantly render the page with that data. Many popular libraries do this but the library I present here allows for this but only includes the minimal code and functionality that is needed to do so. So less code means less network congestion and faster pages.

インターナショナルオープンデータデイ２０１４

Community, Events, Web Development

週末金沢市へインターナショナルオープンデータデイを参加しに行きました。「Code for Kanazawa」という組織が開いてくれました。

「Code for Kanazawa」とは？
「Code for Kanazawa」は市民のための組織です。課題を解決するためにITやデザインの力でソフトウェアやハードウェアを開発します。

イベントは４時間ぐらいでした。最初に三人の発表を見て、六七人のグループを集め、ワクショップの活動をしました。テーマは「住みやすい町ってどんな町でしょうか？」でした。それの中に様々な課題ありますね。課題が数多くあるので、客グループに別で自分たちの課題を考えました。課題を整理・分析した上で、客グループはほかのグループに発表してあげました。

実は三人の発表が全部日本語であまりわからなくてしまいましたが。でも、グループの活動ができたり、ほかの開発者と話す機会ありましたから、とても有意義でした。次のイベントも本当に行きたいです。
